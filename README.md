# Spring Boot Starter Project
> Initial project for any Spring Boot project
### What's in?
This project contains a cool kick off including these dependencies:
- Java ( 15 )
- Lombok ( v. 1.18.20 )
- Jackson ( v. 2.12.3 )
- Spring Boot Web Dependency
- Spring Boog DevTools Dependency

### Run the service
```sh
mvn spring-boot:run
```
